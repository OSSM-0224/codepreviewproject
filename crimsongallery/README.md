# 🎨 Crimson Gallery - React Posting Platform

## 🚀 Project Overview

Crimson Gallery is a modern **React-based posting platform** where users can create, edit, delete, and view posts with images.

This project focuses on **clean UI, user experience, and beginner-friendly React architecture** while implementing core CRUD operations.

---

## ✨ Features

* ➕ Create Post (with image upload)
* ✏️ Edit Post
* ❌ Delete Post
* 📄 View All Posts
* 🖼️ Image Preview before upload
* 📱 Fully Responsive Design
* 🎨 Custom UI (Dark Theme - Crimson Style)

---

## 🛠️ Tech Stack

* React (Vite)
* Tailwind CSS
* React Hook Form
* useState, useRef, useEffect

## 🎯 Why I Chose "Crimson Gallery"

I chose the **Crimson Gallery theme** because I wanted to build something beyond a basic CRUD app.

Instead of a simple blog UI, I designed this project as a **creative exhibition platform**, where each post feels like a piece of art.

* 🎨 Crimson color gives a **premium & artistic vibe**
* 🌑 Dark theme improves **focus on content**
* 🖼️ Posts feel like **gallery entries instead of normal posts**
* 💡 It reflects a more **real-world product design thinking**

This helped me:

* Improve UI/UX thinking
* Build visually appealing layouts
* Understand how design impacts user interaction
